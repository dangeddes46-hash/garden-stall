# 2026-07-02 — Trading controls

Added the first pause/speed/debug control layer on top of the trading checkpoint clock.

Changed:

- Current version is now `0.1.18-trading-controls`.
- Trading clock modes added: Paused, Step, Fast, and Debug instant.
- Trading screen now shows clock-control buttons and the current control note.
- Added `SET_TRADING_CLOCK_MODE` reducer action.
- Added `RUN_REST_TRADING_DAY` reducer action for fast smoke-testing the remaining checkpoints.
- Existing `RUN_TRADING_CHECKPOINT` still runs one checkpoint at a time.
- Trading log entries now record the clock mode used for each checkpoint.
- Clock-control buttons wrap safely using the stock layout override file.

Still not added:

- real unattended timer
- staff implementation
- upgrades
- save/load
- drag-and-drop display
- final art dependency

Notes:

- This is intentionally still a checkpoint-driven simulation.
- Paused mode means no automatic clock movement; manual checkpoint/debug buttons remain available for testing.
- True unattended timer behaviour should only come after this control model is smoke-tested.
