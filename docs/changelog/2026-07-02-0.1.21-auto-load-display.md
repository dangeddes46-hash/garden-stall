# 2026-07-02 — Auto load and auto display

Added player-triggered automation buttons for common stock movement chores.

Changed:

- Current version is now `0.1.21-auto-load-display`.
- Added `prototype/src/systems/autoStockSystem.js`.
- Added `AUTOLOAD_VAN` reducer action.
- Added `AUTODISPLAY_STOCK` reducer action.
- Van Loadout now has an `Autoload varied van` button.
- Display Setup now has an `Autodisplay varied stall` button.
- Autoload prioritises variety first, then older home stock, while respecting tray and feature-pot van capacity.
- Autodisplay prioritises variety first, then newer van stock.
- Autodisplay places higher potential value / giftable / flowering / feature-pot stock into premium display spots where possible.
- Autodisplay can set likely premium-spot stock to Premium pricing.

Still not added:

- auto-ordering
- staff implementation
- upgrades
- save/load
- drag-and-drop display
- final art dependency

Notes:

- These are player-controlled convenience buttons, not AI management or staff behaviour.
- Manual load, unload, placement, price, watering, and reduced-area controls remain available.
