# 2026-07-02 — Evening order feedback

Connected the daily report loop back into the next evening order screen.

Changed:

- Current version is now `0.1.20-evening-order-feedback`.
- Added `prototype/src/systems/eveningFeedbackSystem.js`.
- Wholesaler screen now shows a `Stall notebook` panel before the supplier listings.
- First order shows a balanced-basket reminder.
- Later evening orders show practical clues from the latest daily report.
- Feedback includes largest unsold line, untested leftovers, missed demand warnings, condition warnings, pricing clues, and next-order guidance.
- Feedback is advisory only: it does not auto-fill the cart or prevent any purchase.

Still not added:

- auto-ordering
- staff implementation
- upgrades
- save/load
- drag-and-drop display
- final art dependency

Notes:

- This makes the order/display/trade/report/order loop more coherent without adding a new progression system.
- The intent is to help the player learn from yesterday, not to optimise for them.
