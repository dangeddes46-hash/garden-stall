# 2026-07-05 — Stock handling and summary readability

Small practical pass after the first week-long playthrough.

Current version:

`0.1.25-stock-handling-summary-readability`

Changed:

- Added separate sundry van capacity.
- Van capacity is now 6 tray spaces, 6 feature-pot spaces, and 3 sundry spaces.
- Sundry stock now uses sundry slots instead of tray or feature-pot slots.
- Compost uses 1 sundry slot per bag/batch.
- Terracotta pot packs use 1 sundry slot per 10 units.
- Potted lilies now use grouped feature-pot capacity: 1-3 lilies use 1 feature-pot slot, 4-6 use 2.
- Feature-pot stock creation now supports grouped batches for potted lilies.
- Van schematic now includes a separate sundry bay and shows multi-slot grouped batches.
- Added compatible stock consolidation helper.
- Added reducer actions for home, visible, and all compatible stock consolidation.
- Added `Consolidate home trays` in Van Loadout.
- Added `Consolidate visible trays` in Display Setup and Current Stall.
- Daily Summary `What did not sell` now shows condition-aware leftover wording.
- Daily report packed-stock summaries now include condition counts, dominant condition, and mixed-condition status.
- Markdown export now adds `Seven-Day Breakdown`, `Weekly Accounting`, and `Stock Summary` before raw stock batches.
- Weekly Summary now includes daily breakdown and rough accounting.
- Smoke test now covers sundry capacity, potted lily grouped capacity, consolidation invariants, version string, and the new Markdown sections.

Consolidation compatibility rule:

Compatible batches must share plant id, batch type, batch label, load type, location, display zone, condition, moisture, price band, unit retail price, and reduced flag. Sold, discarded, empty, and out-of-scope batches are not merged.
