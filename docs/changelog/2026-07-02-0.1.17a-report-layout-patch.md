# 2026-07-02 — Report and layout patch

Cleaned up issues found during the 0.1.17 trading-clock smoke test.

Changed:

- Current version is now `0.1.17a-report-layout-patch`.
- Stock tray cards now use explicit tray-detail and tray-action sections.
- Added `prototype/src/stockLayout.css` and imported it after the main stylesheet.
- Stock action buttons now wrap more safely in narrow panels.
- Stock group headings now say `tray` / `trays` and `plant` / `plants` correctly.
- Daily Summary blocker wording now distinguishes stock that never really got a selling test from stock that sold out.
- Daily Summary suppresses the misleading empty-display blocker when the stall sold visible stock earlier in the day.
- Added `prototype/src/systems/reportSystemPatched.js` for safer Markdown report export without rewriting the older report module.
- Debug overlay now uses the patched report exporter.
- Markdown Detailed Condition Changes now falls back to the latest daily report condition events when the live state condition log has moved on.
