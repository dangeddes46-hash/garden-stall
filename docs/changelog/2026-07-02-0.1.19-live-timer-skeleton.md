# 2026-07-02 — Live timer skeleton

Added the first unattended timer layer on top of the trading checkpoint clock.

Changed:

- Current version is now `0.1.19-live-timer-skeleton`.
- Fast mode now auto-runs the next checkpoint every 3.5 seconds while trading is active.
- Debug instant mode now auto-runs the next checkpoint every 0.9 seconds while trading is active.
- Paused and Step remain manual modes.
- Trading screen uses a React effect to schedule and clear the next checkpoint timer safely.
- The auto timer stops when the trading clock reaches packdown-ready.
- The UI shows whether auto advance is active or waiting.
- Manual checkpoint and Run rest of day controls remain available for testing.

Still not added:

- full real-time minute-by-minute simulation
- staff implementation
- upgrades
- save/load
- drag-and-drop display
- final art dependency

Notes:

- This is intentionally still checkpoint-based under the hood.
- The live timer is a small wrapper around the existing checkpoint simulation, preserving fast testability.
- This should be smoke-tested carefully for duplicate checkpoint firing, especially in React Strict Mode.
