# 2026-07-02 — Trading clock skeleton

Added the first trading-day clock layer without replacing the existing testable wave simulation.

Changed:

- Current version is now `0.1.17-trading-clock-skeleton`.
- Added `prototype/src/systems/tradingClockSystem.js`.
- Added a visible 09:00–16:00 trading schedule to the Trading Day screen.
- Mapped the existing passive customer waves onto named time checkpoints.
- Trading log entries now include checkpoint time, title, and description metadata.
- The fifth checkpoint advances to packdown readiness rather than creating a fifth sales wave.
- Trading clock state resets when entering a new trading day, debug-jumping to trading, advancing to next day, or opening the next evening order.

Still not added:

- real unattended timer
- pause/speed controls
- staff implementation
- upgrades
- save/load
- drag-and-drop display
- final art dependency
- new major gameplay systems

Notes:

- This is deliberately a checkpoint clock, not a live timer yet.
- The design keeps fast testing intact while moving the time model toward the intended real-time day structure.
